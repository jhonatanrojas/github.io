## 🚀 Instalación y Uso

### Clonar el repositorio
```bash
git clone https://github.com/jhonatanrojas/jhonatanrojas.github.io.git
cd jhonatanrojas.github.io
```

### Abrir localmente
```bash
# Opción 1: Abrir directamente en el navegador
open index.html

# Opción 2: Usar un servidor local (recomendado)
python -m http.server 8000
# o
npx serve .
```

### Personalización
1. **Actualizar información personal** en `index.html`
2. **Cambiar imágenes** en la carpeta `assets/img/`
3. **Modificar estilos** en `assets/css/modern-light.css`
4. **Actualizar proyectos** y experiencia según necesidades
